CSS

Cascading Style Sheet (CSS) merupakan standar untuk mengatur tampilan yang ada pada suatu berkas HTML. CSS bukan bahasa pemrograman, tetapi merupakan bahasa deklaratif untuk menyatakan tampilan.

JavaScript

JavaScript merupakan bahasa pemrograman yang berjalan di sisi pengguna (client-side) atau di peramban, bukan di server.
